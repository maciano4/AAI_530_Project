# Exploratory Data Analysis (EDA)

This folder contains the data preprocessing and exploratory analysis notebook.

## File

Project_Data_Cleaning_&_EDA.ipynb

The notebook performs:

- Data cleaning and missing value handling
- Timestamp conversion and alignment
- Feature preparation
- Hourly resampling for time-series modeling
- Initial visualization of gas concentration trends

The output of this stage is the cleaned dataset used for model training.
